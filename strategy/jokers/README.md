## 当前总原则

### 基线：先买可负担小丑牌，再用日志修正优先级

- 适用局面：商店出现可购买小丑牌，且没有更精细的单牌记忆。
- 决策规则：早期优先增加稳定得分能力；后续通过失败日志识别低价值购买和错过的强协同。
- 协同：暂未细分。
- 反例：如果小丑牌占位紧张、现金不足或已有收益冲突，盲目购买可能降低长期收益。
- 证据来源：`ShopAgent` 当前对 `JOKER` 给出高于消耗牌和补充包的基础分。
- 待验证问题：哪些小丑牌应被标记为早期优先、过渡、后期核心或避免购买？

### 工作假设：优先选择成长型小丑牌

- 适用局面：商店出现能随出牌、弃牌、消耗牌、购买/出售、牌型次数或回合行为持续变强的小丑牌，且当前局面仍有足够回合让成长收益兑现。
- 决策规则：同等价格和生存压力下，优先买成长型小丑牌，而不是只提供固定小额筹码/倍率的小丑牌。小丑槽满时，若成长型小丑牌的长期收益明显高于过渡牌，应考虑卖出低成长或低协同小丑牌腾槽。
- 协同：成长型小丑牌越早拿越好；后续商店和手牌决策应围绕它的触发条件组织，例如优先使用能增强它的消耗牌、选择能触发它成长的出牌方式，或保留能稳定触发成长条件的牌型材料。
- 反例：如果当前盲注已经面临死亡压力，不能为了未来成长牺牲本回合最低得分；如果成长条件和当前核心小丑冲突，也不能盲目追成长。
- 证据来源：用户在 2026-06-01 提供的策略 know-how；仍需后续真实运行日志验证。
- 待验证问题：哪些具体小丑牌应进入“成长型优先名单”，以及每张牌对应的最佳增强操作是什么？
