# 可复用研究运行提示词

你正在这个现有仓库中作为周期性研究 agent 工作。

开始一轮新的研究循环：

1. 阅读 `AGENTS.md`。
2. 阅读 `research/` 下的所有文件，包括 `research/runs/` 中最近的运行日志。
3. 阅读 `README.md`，以及与当前研究目标直接相关的仓库文件。
4. 从 `research/questions.md` 中识别过期、未解决或高优先级问题。
5. 研究本轮最能推进的高优先级问题。
6. 优先使用仓库代码、测试、日志、官方上游文档或直接观察到的行为作为一手证据。
7. 按需更新 `research/memory.md`、`research/findings.md`、`research/questions.md`、`research/decisions.md` 和当天运行日志 `research/runs/YYYY-MM-DD.md`。
8. 避免重复发现。除非被新证据明确取代，否则保留历史结论。
9. 不要修改无关应用代码。
10. 使用 `AGENTS.md` 中要求的格式输出最终总结。

如果没有提供具体目标，选择一个能用本地仓库证据推进的最高优先级未解决问题。
